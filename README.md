# Chatbot Frontend (React + TypeScript)

This project is a chatbot frontend built with React and TypeScript.

The goal of this app is to practice clean component structure, TypeScript typing, async message handling, and a smooth chat UI experience.

## What This Project Includes

- React + TypeScript frontend architecture
- Reusable chat components (input, messages, message item)
- Async message flow with loading state while waiting for response
- LocalStorage persistence for chat history
- Auto-scroll to keep the latest message in view

## Tech Stack

- React
- TypeScript
- Vite
- Day.js
- CSS

## Frontend and Backend Note

This is mainly a frontend project (React + TypeScript).

It also includes a small backend-like flow in the app logic through async request/response handling, so the UI already behaves like a real chat client.

## Run Locally

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
